# docker-gunicorn-meinheld
A Docker image for running Python web apps with Gunicorn + Meinheld workers (for big speed)

## Environment variables
`ROOT_DIRECTORY`
`APP_SUBDIRECTORY`

`PREFLIGHT_PATH`
`PREFLIGHT_FILE`

`GUNICORN_CONF_PATH`
`GUNICORN_CONF_FILE`

`REQUIREMENTS_FILE`

`PYTHON_MODULE_NAME`
`PYTHON_VARIABLE_NAME`
`PYTHON_APP`

`HOST`
`PORT`
`BIND`

`WORKERS_PER_CORE`
`LOG_LEVEL`
